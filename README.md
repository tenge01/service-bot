# Бот для сервисных заявок школ

## Файлы проекта
- `bot.py` — основной код бота
- `requirements.txt` — библиотеки
- `Procfile` — команда запуска для Railway
- `data/services.xlsx` — ваша таблица сервисных центров (загрузите сами)

## Формат таблицы services.xlsx
Первая строка — заголовки, далее данные:

| Город | Тип техники | Бренд | Сервис | Контакт |
|-------|------------|-------|--------|---------|
| Астана | Принтер | HP | ТОО Принт Сервис | +77001234567 |
| Алматы | МФУ | Canon | Canon Service | +77771234567 |

## Команды для администратора (вам в Telegram)
- `/tickets` — последние 10 заявок
- `/status ZV-20240101-0001 готово` — сменить статус заявки

## Деплой на Railway
1. Загрузите все файлы в GitHub репозиторий
2. В Railway: New Project → Deploy from GitHub
3. Добавьте переменные окружения:
   - BOT_TOKEN = ваш токен
   - ADMIN_CHAT_ID = ваш Telegram ID
